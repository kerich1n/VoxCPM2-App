# 🎙️ VoxCPM2 語音克隆 App — 安裝說明

## 怎麼使用

1. 打開 App，點上方「**人物管理**」分頁
2. 填人物名稱 → 上傳一段參考音檔（建議 **10～15 秒**、清楚、無雜音）→ 填那段音檔的逐字稿（強烈建議填，能改善效果）→ 按「**新增這個人物**」
3. 回「**生成語音**」分頁 → 選擇人物 → 輸入要念的台詞 → 按「**生成**」
4. 生成的聲音會出現在畫面上，可播放，檔案也會存到 `VoxCPM2-App/outputs` 資料夾

---

## 常見問題

**Q：生成的聲音怪怪的？**
試試把參考音檔換長一點（10～15 秒）、確認逐字稿跟音檔內容一致。進階設定裡也可以調整參數。

**Q：第一次打開說「無法打開，因為來自未識別的開發者」？**
不要直接雙擊，改成**對 App 按右鍵 → 打開**，再點「打開」即可。
